Что такое функция?

  1. Часть кода, которому мы даём имя
  2. Функцию можно написать один раз и вызывать из разных мест программы
  3. Функции помогают структурировать код, разбив его на отдельные действия
  4. Делают код более читабельным и упрощают поиск ошибок

Представим программу как конвеер

Предметы на ленте - переменные

Работу выполняют функции

Что нужно знать о функциях

  • Создаем функцию при помощи ключевого слова def, после которого пишем имя функции
  • Узнать функцию просто - после названия функции идут круглые скобки
  • В название функции используйте маленькие латинские буквы, цифры, подчеркивания: get_user_name() или open_workbook()

Доступно много функций "из коробки"

Функция print() распечатывает то, что передали ей в скобочках.

>>> print('Привет!')
Привет!

Доступно много функций "из коробки"

Функция len() возвращает длинну того, что передали ей в скобочках.

>>> len('Привет!')
7

Доступно много функций "из коробки"

Функции можно объединять:

>>> print(len('Привет!'))
7

Разберемся, зачем нужны функции

Создайте в Sublime файл test_func.py

price = 100
vat_rate = 18

vat = price / 100 * vat_rate

print(vat)

Усложняем

price = 100
vat_rate = 18

vat = price / 100 * vat_rate

price_no_vat = price - vat
print(price_no_vat)

Можно сказать, что задача решена. Но в реальной жизни такое решение не подходит. Почему?

Создадим функцию

def get_vat(price, vat_rate):
    vat = price / 100 * vat_rate
    price_no_vat = price - vat
    print(price_no_vat)

price1 = 100
vat_rate1 = 18
get_vat(price1, vat_rate1)

Теперь попробуем использовать функцию для разных задач

price1 = 100
vat_rate1 = 18
get_vat(price1, vat_rate1)

price2 = 500
vat_rate1 = 10
get_vat(price2, vat_rate2)

get_vat(50, '5')
get_vat(-100, 18)

Как нам починить функцию?

Аргументы

  • Функция — это черный ящик в который мы можем что-то положить, а на выходе получить результат.
  • Аргументы — данные, с которыми мы хотим работать внутри функции.
  • Названия аргументов мы придумываем сами, ограничения на названия — как у обычных переменных.

Позиционные аргументы

Проверочное слово "первый аргумент", "второй аргумент".

def get_summ(one, two):
    return one + two

Позиционные аргументы

get_summ(2, 2)
???

get_summ('Hello', 'world!')
???

Именованные аргументы

Задаем название и значение аргумента.

def get_summ(one, two, delimeter=' '):
    return str(one) + str(delimeter) + str(two)

Проверим

get_summ('Hello', 'world!', delimeter='&')
???

get_summ('Hello', 'world!', ' + ')
???

get_summ('Hello', 'world!')
???
