Условный оператор

Выполняет разный код в зависимости от данных.

Условный оператор

			if age < 18:
			    print('Никаких тебе сигарет. Ишь чего удумал!')
			else:
			    print('Вам синий или красный?')

Условный оператор

			if not is_user_banned:
			    ban_user(user)

Условный оператор

			if team == 'mystic':
			    print('Hi, blue team')
			elif team == 'instinct':
			    print('Hi, yellow team')
			elif team == 'valor':
			    print('Hi, red team')

Условный оператор

			if team == 'mystic':
			    if user_level >= 5:
			        print('Welcome to the gym')
			    else:
                    print('Get more exp')

Условный оператор

  • Нужен, если нужно что-то сделать при каком-то условии.
  • Если условие истинно, то выполняется тело функции.
  • Если условие ложно, выполняется ветка else (если она есть).
  • Можно последовательно проводить много проверок с помощью elif.
  • Можно вкладывать один условный оператор в тело другого.

Сравнение строк

  • Написать функцию, которая принимает на вход две строки.
  • Если строки одинаковые, возвращает 1.
  • Если строки разные и первая длиннее, возвращает 2.
  • Если строки разные и вторая строка 'learn', возвращает 3.
